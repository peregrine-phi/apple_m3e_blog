## ToC 下拉菜单液态玻璃效果修复
- 移动端 `.toc-nav-mobile` 背景从不透明 surface 切回 liquid glass，并移除有色外边框
- 根因：`.toc-nav-mobile` 是 `.toc-mobile`（details）的子元素，嵌套 backdrop-filter 导致内外层模糊叠加、效果不一致
- 解法：重构 DOM，`.toc-nav-mobile` 提出为 `.toc-mobile-wrapper` 的直接子元素，与 `<details>` 平级，各自独立 backdrop-filter
- JS 同步 `<details open>` → `.open` class 控制下拉显隐
- 下拉菜单链接查询从 `mobileToc.querySelectorAll` 改为 `wrapper.querySelectorAll`
- 追加点击外部关闭功能
